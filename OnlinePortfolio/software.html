High Statistics Data Analysis

CCDeltaPlus Reconstruction Package

About:
Process raw data taken by MINERvA Detector to identify and reconstruct the exclusive channel:

ν μ + {n} C → μ - + p + π 0

Capabilities:
  • Part of MINERvA Framework
  • Identifies Δ+ Production Signal in raw data, by detecting each final state particle
  • Reconstructs each particle trajectory and calculates its 4-Momentum (Px,Py,Pz,E)
  • Using reconstructed particle information estimates interaction kinematics
  • Creates output file which is read by a Data Analysis Package for a more detailed data analysis
Technologies:
  • C++
  • ROOT Data Analysis Framework
  • Shell Script

Neutrino Oscillations MATLAB Toolbox

About:
MATLAB functions enhanced with GUI used to simulate neutrino oscillations in matter. Simulation results may help existing and future neutrino oscillations experiments.
Capabilities:
  • Calculates Neutrino Oscillation Probability
  • Calculates Event Rate for Nova
  • Detailed analysis of event kinematics and statistics with different background signals
